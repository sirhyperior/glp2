## STEP 3 — Git 설치

**Windows**
```
윈도우 키 + R → cmd 입력
winget install Git.Git
약관 동의 Y → Y (2번)
cmd 닫고 다시 열기
git --version 확인
```

**Mac**
```
Cmd+Space → terminal 열기
brew --version 확인 (없으면 Homebrew 먼저 설치)
brew install git
git --version 확인
```

**초기 설정 (공통)**
```bash
git config --global user.name "김북곽"
git config --global user.email "your@email.com"
```

## STEP 4 — 터미널에서 첫 커밋

Git 설치 후 터미널에서 직접 쳐보세요:

```bash
git clone https://github.com/내아이디/my-first-repo.git
cd my-first-repo
# 파일 수정 후
git add .
git commit -m "터미널에서 첫 커밋!"
git push origin main
```

웹에서 한 것과 똑같은 일이지만, 다양한 방법을 익혀보아요.
